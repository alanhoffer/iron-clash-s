Documentación para generación de assets estilo “La Brute”
1. Estilo visual general

Todos los assets deben mantener el mismo estilo para que encajen en el juego:

Estilo: Cartoon / humorístico, inspirado en La Brute (My Brute)

Proporciones: Exageradas y divertidas (personajes con cabeza grande, muebles con patas exageradas, armas grandes y cómicas)

Vectorial: Formas limpias y definidas, estilo vector art

Contornos: Negros y gruesos

Colores: Planos con cel shading (sombras simples, mínimo gradiente)

Estética: Flash animation, divertido, expresivo, fácil de reconocer

Fondo: Blanco o transparente según el asset

Resolución: Alta, game-ready

2. Personajes
Características

Tipo: Humano o criatura

Estilo: “Funny cartoon gladiator” / “humorous warrior”

Proporciones: Cabeza grande, manos y pies exagerados

Pose: Neutral o T-pose, listo para animación cutout

Partes separadas para animación (opcional):

Head

Torso

Left Arm / Right Arm

Left Leg / Right Leg

Weapon / Accessory

Prompt ejemplo
2D game character sprite, style of "La Brute" (My Brute) web game,
vector art style, thick black outlines, flat colors, cel shading,
funny cartoon warrior, exaggerated proportions (big head, expressive face),
dynamic idle pose or neutral stance,
parts separated for cutout animation: head, torso, left arm, right arm, left leg, right leg, weapon,
white background, flash animation aesthetic, humorous and tough vibe

3. Muebles
Características

Tipo: Medieval o moderno según nivel/tema del juego

Vista: Frontal (front view), estática para sprites 2D

Partes separadas para animación (opcional): puertas, cajones, tapas, cojines

Ejemplos:

Wooden throne, table, chair, barrels, shelves

Treasure chest, medieval lamp, bookshelf

Prompt ejemplo
2D game medieval furniture sprite, style of "La Brute" (My Brute),
vector art style, thick black outlines, flat colors, cel shading,
funny cartoon medieval furniture with exaggerated playful proportions,
front view only, static pose suitable for 2D game,
items such as wooden throne, treasure chest, medieval table, shelves, barrels, candles,
white background, clean vector shapes, flash animation aesthetic, humorous and charming vibe

4. Armas
Características

Tipo: Espadas, mazas, hachas, arcos, lanzas

Estilo: Exagerado, cómico, reconocible a simple vista

Colores: Planos con sombras simples

Partes separadas para animación (opcional): hoja / mango, guardas, accesorios

Prompt ejemplo
2D game medieval weapon sprite, style of "La Brute" (My Brute),
vector art style, thick black outlines, flat colors, cel shading,
funny cartoon sword/axe/mace with exaggerated proportions,
front view only, static pose, clean vector shapes,
white background, flash animation aesthetic, humorous and charming vibe

Características

Estilo: Exagerado y divertido, manteniendo proporciones caricaturescas

Partes separadas para animación: casco, torso, brazos, piernas, accesorios

Colores: Planos con detalles mínimos, cel shading

6. Tips generales para todos los assets

Consistencia: Mantener contornos, colores y estilo uniforme.

Exageración: No tener miedo de hacer cabezas, armas o muebles grandes y cómicos.

Separación de partes: Siempre separar los elementos que podrían necesitar animación.

Frontal view: Para sprites de juego 2D, todos los assets deben estar de frente.

Exportación: PNG con fondo transparente o fondo blanco si es necesario para IA.
